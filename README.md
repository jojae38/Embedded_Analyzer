# Embedded Analyzer

## Introduction

This is a tool to analyze embedded software. It is designed to be used with the [Embedded Analyzer](https://github.com/jojae/Embedded_Analyzer) and [Embedded Analyzer GUI](https://github.com/jojae/Embedded_Analyzer_GUI).

## Installation

### Windows

1. Download the [latest release](https://github.com/jojae/Embedded_Analyzer/releases/latest).
2. Extract the contents of the zip file.
3. Run the `EmbeddedAnalyzer.exe` file.

# WebUI

npm install
npm run dev

# Build

npm run build
